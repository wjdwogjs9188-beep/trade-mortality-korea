**v3.2 narrative 잘못**: "외국인 비율의 시군구별 heterogeneity 가 manufacturing 비중과 상관 → measurement error × treatment 상관 → attenuation bias 또는 differential measurement error."

**v3.3 정확화**:

> "Panel 의 인구 분모 (KOSIS DT_1B040M5 주민등록연앙인구) 가 사실상 **한국인 only** (행안부 한국인 only 와 -0.35% 차이, Stage 3B v02.1 verify). 사망 분자도 외국인 제외 (is_foreign_or_unknown=False 필터, Stage 2 v4 처리). 즉 분자/분모 모두 한국인 only 일관.
>
> 따라서 v3.2 의 'measurement error × treatment 상관' 우려는 부분 잘못. 외국인 비율의 시군구 heterogeneity 는 measurement error source 가 아니라 (분모/분자 모두에서 외국인 제외됨) **분석 sample 의 정의 차이** source. 즉 manufacturing-heavy 시군구 (안산, 시흥, 화성) 의 외국인 노동자가 분석에서 제외 → main 결과 = 한국인 only 의 mortality 영향만.
