### 5.2 Mediation analysis (v3.2: DGHP 2017 application + DFH 2020 Stata methodology 분리 인용, framework mechanics 정확화)

**v3.2 정확 framework — 두 paper 분리 인용**:

1. **Dippel, Gold, Heblich, Pinto (DGHP) 2017, "Instrumental Variables and Causal Mechanisms: Unpacking the Effect of Trade on Workers and Voters," NBER Working Paper 23209** — Application paper (독일 1987-2009 trade-voting mediation framework 의 first introduction)
2. **Dippel, Ferrara, Heblich (DFH) 2020, "Causal mediation analysis in instrumental-variables regressions," Stata Journal 20(3): 613-626** — **Methodology paper + Stata `ivmediate` package** implementation 표준

본 paper 는 두 paper 를 모두 인용. DGHP 2017 = framework 의 first introduction (trade-mediator-outcome). DFH 2020 = 본 paper 가 사용할 Stata `ivmediate` implementation 의 reference.

**DFH 2020 정확 식별 조건 (v3.2 verification 후 수정)**:

- **Core assumption**: **ρ_TY = 0** — T (treatment) 와 Y (outcome) 간 unobserved confounder 부재
- **Endogeneity 의 source**: T 의 endogeneity 는 (T-Y 채널) 에서가 아니라 **(T-M 채널)** 에서만 발생 가능 — 즉 Z 가 T 와 M 모두에 대한 valid IV
- 이 가정 하에서 Z 를 **M 의 IV 로 사용 가능 (T conditional 후)** — additional instrument for mediator 불필요

**DFH 2020 정확 estimation procedure (v3.2 verification)**:

3 separate 2SLS regressions:
1. **T → M** (β_TM): Z 를 IV 로 T 추정 후 M 에 대한 effect
2. **T → Y** (β_TY = total effect)
3. **Y → M conditional on T** (γ_MY|T): T conditional 후 M 의 Y 에 대한 effect

산출:
- **Indirect effect** = β_TM × γ_MY|T
- **Direct effect** = β_TY − Indirect = β_TY − (β_TM × γ_MY|T)
- **Total effect** = β_TY (standard IV regression coefficient)
- **Mediation share** = Indirect / Total

**v3.1 의 부정확 narrative 수정**:
- v3.1 commit "1 IV 로 3 effects 직접 식별" → 부정확. **3 separate 2SLS regressions** 로 indirect/direct/total decomposition 이 정확.
- v3.1 commit 식별 조건 4가지 (IV exclusion, U_M ⊥ U_Y | X, sequential effect, no unobserved direct effect heterogeneity) → 부정확. **ρ_TY = 0 (T-Y unobserved confounder 부재)** 가 core assumption.

**v3.2 honest framing (DFH 2020 의 strong assumption)**:

> "DFH (2020) framework 의 core assumption ρ_TY = 0 은 T 와 Y 간 unobserved confounder 부재를 요구한다. Bartik IV setting 에서 baseline industry shares 가 trade shock T 와 mortality outcome Y 에 동시 영향하는 unobserved confounder (예: 시군구별 인구 구조, 의료 인프라, 외국인 비율) 가 존재하면 ρ_TY ≠ 0 → indirect effect 추정 bias. 본 paper 는 § 4 의 baseline controls (manufacturing share, education, age structure, urbanization, sido FE, 의료 인프라, 외국인 비율) 추가로 conditional ρ_TY = 0 을 maximally 확보하지만, 가정 위반 가능성 명시. Mediation 추정 결과의 quantitative magnitude 는 strong assumption 하의 best estimate."

**Sensitivity**:

**v2 의 "MDE 7-8배" calibration error 수정**:

**Pierce-Schott (2020, AER:I) 비교 baseline**:
- Cross-sectional units: **3,122 county**
- Sample size: ~74,924 (county × year)
- Effect size: β ≈ 0.5-1.5

- Cross-sectional units: **229 시군구** (Pierce-Schott county 의 1/13.6)

**SE inflation first-order approximation** (cross-sectional units 기준):

```
SE_(본 paper) / SE_(Pierce-Schott) ≈ √(3,122 / 229) = √13.6 ≈ 3.7
```

**Implication (v3.1 수정 — logical error 제거)**:

본 paper 의 cross-sectional effect size 는 **사전에 알 수 없다**. Pierce-Schott 의 미국 effect (β ≈ 0.5-1.5, suicide IV coefficient) 가 한국에서 동일하다고 *가정* 시, SE inflation ~3.7배로 detection 가능성 borderline. Null finding 시 (a) Korean effect 가 본 paper detectable threshold 미만 (b) Korean effect 가 미국과 차이, 두 가능성을 본 paper 단독으로 구분 불가.

13. **Weak-IV test 의 instrument-specific 가정** (N2): OP 2013 effective F 임계값 23.1 은 single endogenous + single IV 케이스. 본 paper 가 8 OHIE 합산 IV 로 single-IV 처리하지만 sensitivity 로 OHIE 개별 IV 분리 사용 시 multi-IV 임계값 확인 필요.
