# [#25787] Can Economic Policies Reduce Deaths of Despair?

## 메타정보
- **저자**: William H. Dow (UC Berkeley SPH), Anna Godøy (UC Berkeley/Statistics Norway), Christopher A. Lowenstein (UC Berkeley), Michael Reich (UC Berkeley/NBER)
- **년도**: 2019년 4월, 2020년 8월 revised
- **학술지**: NBER Working Paper No. 25787
- **JEL**: I1, I38

## Research Question + Contribution

**경제 정책(최저임금, EITC)이 "절망의 죽음(deaths of despair)" - 약물, 술, 자살 - 을 인과적으로 감소시킬 수 있는가?**

핵심 기여:

1. **인과 식별**: 주(state)별 정책 변동의 시간 변화를 활용한 준실험 설계
2. **자살에 제한된 효과 발견**: 약물/알코올 사망에는 유의미한 감소 없음, **비약물 자살만 감소**
3. **성별 이질성**: 정책 효과가 여성에서 더 큼 (노출도와 일치)
4. **건강-경제 정책의 새로운 증거**: 기존 불명확한 관계(소득 증가 → 약물 사용 증가 vs 감소)를 메커니즘별로 구분

## Data

### 사망 데이터 (기본 분석):
- **출처**: CDC Multiple Causes of Death (제한 접근 geocoded)
- **기간**: 1999-2017 (일관된 부호화)
- **표본**: 18-64세 (비고령자), 제외 state 4개 (education data 불완전)
- **결과**: 46개 주 + DC

### 사망 원인 분류 (ICD-10):
1. **약물 비자살(Unintentional drug OD)**: X40-X44, Y10-Y14
2. **약물 자살(Drug suicide)**: X60-X64
3. **비약물 자살**: X66-X84, Y870
4. **알코올**: K70, K73-K74, X45, X65, Y15 (간질환 + 중독 중독)
5. **대조(Placebo)**: 암(C00-C96)

- **주요**: 고졸 이하 (정책 노출 높음)
- **통제**: 학사 이상 (정책 노출 낮음, placebo)
- **이질성**: 성별, 인종/민족별 분리 분석

### 보조 데이터:

- **UKCPR**: GDP, SSI 점유율, 실업률, EITC 정책 시간표
- **추가**: Medicaid 확대, 의료용/오락용 마리화나 합법화, PDMP(처방약 모니터링)

### 처리 변수:
1. **최저임금**: max(연방, 주 수준), 2016년 달러 기준
   - 변동: 1999-2017 주(state) × 연도별 광범한 변동
2. **EITC**: 최대 부양자 2인 기준 로그값

   - 16개 주가 2000-2014년 새 EITC 도입 (또는 확대)

## Identification Strategy

### 설계: Generalized Difference-in-Differences (DID)

**기본식**:
$$y_{ist} = \theta_t + \theta_s + X_{ist}\beta_X + \log(mw_{ist})\beta_{mw} + \log(\text{EITC}_{ist})\beta_{EITC} + \varepsilon_{ist}$$

- $\theta_t$ = 년도 고정 효과 (국가 시간 추세)
- $\theta_s$ = 주(state) 고정 효과 (지속적 주 차이)
- $X_{ist}$ = 시간 변화 통제:
  - 나이/성별 지표
  - 보험 미가입률, 주 GDP, SSI 점유율, 인구
  - 실업률, Post-ACA Medicaid 확대, 의료용/오락용 마리화나, PDMP
  - **인종/민족 × 년도 상호작용** (추세 조정)
  - **교육 × 년도 상호작용** (교육 구성 변화 통제)

### 식별 가정: Parallel Trends

**조건**: 정책 변경이 미관측 사망률 추세와 무상관

**위협**:
- 정책이 내생적: 고 최저임금 주는 지리적으로 클러스터되고, 민주당 주, 노조화됨
- 정책 번들: EITC + Medicaid 확대가 동시 시행
- 주 경제 기본: 저 최저임금 주와 다른 추세

**대응**:
1. **주 고정 효과**: 시간 불변 이질성 제어
2. **사건 연구(Event Study)**: 정책 시점 전후 추세 검사
3. **위약 검사(Placebo)**: 대학 졸업생 (정책 노출 낮음), 암 사망(정책 효과 없어야 함)

### 사건 연구 사양:

**기본**:
$$y_{ist} = \theta_t^{pol} + \theta_s^{pol} + X_{ist}^{pol}\beta^{pol} + \sum_{k=-5, k\ne-1} \pi_k(s,t) \rho_k^{pol} + \varepsilon_{ist}$$

**선호 사양** (이질적 처리 크기 조정):
$$y_{ist} = \theta_t^{pol} + \theta_s^{pol} + X_{ist}^{pol}\beta^{pol} + \sum_{k=-5, k\ne-1} (\pi_k(s,t) \times \delta_s^{pol}) \rho_k^{pol} + \varepsilon_{ist}$$

여기서 $\delta_s^{pol}$ = 이벤트 창의 총 정책 변화

- 참고 범주: $k = -1$ (정책 변경 직전)
- 사전 추세 검사: $k < -1$에서 계수 0 근처여야 함
- 단절(Discontinuity): $k = 0$ (최저임금), $k = 1$ (EITC, 1년 지체)

## Main Findings

### 1. 사건 연구 (Event Studies)

**최저임금** (Figure 1a):
- **비약물 자살**: 
  - 사전 추세: 작고 0 근처 (parallel trends 지지)
  - $k=0$에서 유의미한 하강 (정책 시점에 불연속 강하)
  - 사후: 지속적 감소
- **약물/알코올**: 명확한 효과 부재, 오류 추세 문제

**EITC** (Figure 1b):
- **비약물 자살**: 
  - 사전 추세: 평행 (양쪽 성별)
  - $k=0$에서 작은 음의 효과, $k=1$에서 더 큼 (환급 지체 일치)
  - 정책 시행 1년 후 본격 효과
- **약물/알코올**: 명확한 시간 경로 없음

**성별 차이** (Figure 2):
- **여성**: 최저임금 사전 추세 작음, $k=0$에서 통계적 유의성 있는 강하
- **남성**: 추세 덜 명확, 효과 0 근처 (사건 연구), 차분차분은 작지만 음수
- **EITC**: 성별 모두 병렬 추세, 효과 경로 약간 다름 (여성 더 빠름)
- **위약**: 대학 졸업생, 암: 효과 부재 (설계 신뢰도 높음)

### 2. 차분차분(Generalized DID) 결과 (Table 2)

**주 표본 (고졸 이하)**:
- **최저임금**: 10% 증가 → 비약물 자살 **2.7% 감소**
  - $\beta = -0.027, p < 0.05$ (column 3)
- **EITC**: 10% 최대 신용 증가 → 비약물 자살 **3.0% 감소**
  - $\beta = -0.030, p < 0.05$
- **약물/알코올**: 유의미하지 않음
- **알코올**: 최저임금에서 한계 양의 효과 (5%, $p < 0.10$), 이벤트 연구에서 미검증 → 의심

**위약 표본 (학사 이상)**:
- 모든 원인에서 효과 부재 (설계 타당성 지지)

### 3. 성별 이질성 (Table 2, panels C-D)

**여성** (고졸 이하):
- 최저임금: 10% 증가 → **3.4% 자살 감소** (상대적으로 큼)
- EITC: 10% 증가 → **4.1% 감소** (통계적 불확실)
- 표본 크기 작아 정밀도 낮음

**남성** (고졸 이하):
- 최저임금: **1.7% 감소** (여성의 절반)
- EITC: **1.3% 감소**
- 덜 정밀함

**해석**: 여성이 최저임금 일자리/EITC 대상에 더 많이 노출 (Appendix Figure A11)

## Robustness

### 1. 다중가설 검사(Multiple Hypothesis Testing, MHT)
4개 사망 원인 × 2개 정책 = 8개 검사

Appendix Table A3:
- **최저임금 → 비약물 자살**: 보정 후 **1% 수준에서 유의** (robust)
- **EITC → 자살**: 보정 후 $p = 0.155$ (한계, 부정확)

### 2. 선형/이차 주 추세
Table 3:
- 선형 추세 추가: 자살 효과 유의미 (1%)
- 이차 추세: 계수 안정적
- 약물/알코올: 여전히 부정확

### 3. 인종/민족 분리 (Table 4)
- **최저임금**: 백인 hispanics/다른 민족 간 차이 부재 (낮은 통계력)
- **EITC**: 유색인 그룹에서 효과 더 크지만 정밀도 낮음
- **놀라운 음성**: 비백인이 저임금 일자리에 더 노출 → 효과 더 클 것으로 예상했으나 미검증

**해석** (저자): 흑인이 자살 절망감(hopelessness)과 약한 연관 → 공공보건 부등식의 역설적 예

### 4. 대체 명시:
- **사망률** (100,000당): 자살 효과 약간 작음 (1.8%)
- **포아송**: 비선형, 효과 1.3%
- **사건 표본에서 DID**: 주 추세 + 전체 표본과 유사
- **정책 매개변수**: 로그 vs 수준 vs 지시자 → 정성적 유사

### 5. EITC 환급성(Refundability):
환급 가능 vs 불가능 EITC:
- 효과는 환급 가능에서 약간 더 큼
- 지적: 환급은 즉시 현금(short-term shock), 불가능는 세금 인센티브(장기)
- 자살 감소의 즉시 효과(정책 시행 1년 후) → 현금 유동성 메커니즘 시사

### 6. 고용 효과 검사 (Appendix Table A1, Figure A1):
- **최저임금의 고용 효과**: 고졸 이하 표본에서 0 근처 (통계적 무의미)
  - 젊은 여성(25-29): 양의 고용 효과
  - 부정적 고용 효과 없음
- **EITC**: 남성에서 약간 음수 추정 (해석 신중)
- **결론**: 자살 감소는 소득 증가 + 가능한 고용 개선의 혼합 (고용 손실 제외)

## Heterogeneity

### 성별 (주요 발견):
- 여성: 정책 노출 + 반응성 더 큼
- 남성: 약한 반응 (자살 절망감과의 연결 약할 가능성)

### 나이 (암시):
- 자살 효과가 젊은층(18-64)에 집중되는 것으로 보임
- 장기 효과(약물 중독, 알코올 중독의 오랜 누적)는 측정 어려움

### 인종 (역설):
- 정책 노출이 높은 비백인에서 효과 미검증
- 정신건강 메커니즘(절망감 vs 스트레스)의 인종별 차이 추측

## Mechanism

직접 메커니즘 분해 제한. 간접 증거:

### 소득 효과:
- 최저임금 증가 → 저임금층 소득 상승 (문헌)
- EITC → 세금 환급을 통한 현금 증가
- 통제: 고용 효과 검사 (대부분 0) → 순 소득 증가

### 부채/신용 채널:
- 인용 Cooper et al. (2019): 높은 최저임금 → 신용 점수 개선 → 부채 비용 감소
- 자살 유발인 중 가정 재정 위기, 주택 저당 손실 포함 (Reeves 2012, Chang 2013)

### 보험 접근:
- 논문은 제외했지만 관련:
  - EITC 확대 → 소수 의료 보험 획득 → 정신건강 치료 증가
  - 최저임금은 고용과 관계 없으므로 보험 미기반 → 직접 접근 불가

### 미측정 채널:
- 심리사회적: 경제적 체면, 가족 스트레스 감소
- 사회적: 자살 낙인 감소, 혼인/출산율 개선
- 약물: 알코올/약물은 정상재(normal goods) → 소득 증가시 사용 증가 가능(상쇄 가능)

**논문 결론**: 자살은 특이하게 경제 절망과 연결, 약물/알코올은 더 복잡한 관계 (중독, 공급 요인, 가격)

## Quality Assessment (교훈 3개)

### 교훈 1: 위약 및 사건 연구의 보완적 설계
**문제**: DID의 parallel trends 가정이 불확실할 때

**Dow et al. 해결책**:
- 위약 변수: 대학 졸업생 (정책 노출 낮음) → 효과 0 검증
- 사건 연구: 정책 시점 전후 추세 직시 검사
- 대조 결과: 암 사망 → 단기 정책 효과 없어야 함

**본 PAP 적용**:
- 위약 1: 비제조업 지역 (무역 충격 노출 낮음) → 효과 0
- 위약 2: 수입 경쟁 지역 (반대 방향, 고용 증가) → 효과 역부호
- 사건 연구: 무역 협상 시작/타결 시점 전후 분석
- 암: 사망률 비교 (정책과 무관해야 함)

### 교훈 2: 메커니즘의 간접 측정
**한계**: Dow et al.도 고용 효과 외에 직접 메커니즘 측정 못함

**전략**:
- 보조 데이터(CPS)로 정책 노출 검증 (소득 실제 증가? 고용 유지?)
- 환급성(Refundability) 분석: 현금 유동성 vs 세금 인센티브 구분
- 성별 차이: 정책 노출도와 효과의 상관성 검증 (→ 노출-반응 일치)

**본 PAP 적용**:
- 지역별 무역 충격 크기 vs 사망률 변화의 선형성 검사
- 산업별 구조 변화 강도 vs 고용 감소의 추적
- 가족 구조(결혼, 자녀)가 중보 변수인지 탐색 (선택적)

### 교훈 3: 다중검사(MHT)와 결과 안정성
**발견**: EITC → 자살은 MHT 보정 후 $p = 0.155$ (유의 상실)

**인사이트**:
- 다양한 원인(약물, 알코올, 자살)을 검사하면 거짓 발견 위험 ↑
- Bonferroni는 보수적 (독립성 가정 때문에)
- **최저임금 → 자살은 robust** (1% 수준)

**본 PAP 적용**:
- 사망 원인별 영향(자살 vs 약물 vs 알코올) 다중검사
- 산업별 이질성(제조업 vs 서비스) 검사
- 성별/연령 상호작용 검사
- → Romano-Wolf 또는 FDR 보정 사용 권장
