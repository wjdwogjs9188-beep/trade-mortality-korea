## § 5. Main Empirical Results

### 5.1 Reduced-form main estimate

Table 1 reports the long-difference reduced-form estimate of mortality on Korea-China bilateral trade exposure for the n=222 main analytic sample, drawn from the 251-sigungu panel universe (Section 3.2 Table A). The dependent variable is the log change in age-standardized mortality rate (deaths-of-despair composite, working-age 25-64 Korean nationals) between the 1997-1999 base period and the 2018-2022 endpoint period. The regressor of interest is the standardized 1994-baseline Bartik exposure measure z_x_h, constructed as detailed in Section 4.1.

The main estimate is

  β_main = -0.127 (cluster-province t = -4.02; AKM-proper t = -4.93, p_AKM < 10⁻⁶),

indicating that a one-standard-deviation increase in Korea-China bilateral trade exposure reduces the long-difference change in log working-age deaths-of-despair mortality by 0.127 (in log-rate units) over the 1997-2022 period. The estimate is highly significant under multiple cluster-robust inference layers. The Adão-Kolesár-Morales (2019) shift-share standard error, computed using the Kolesár (2024) `ShiftShareSE` R package, yields SE_AKM = 0.0258 with t-statistic -4.93 and p-value < 10⁻⁶ under the AKM (industry-aggregated) inference framework. The Wild Cluster Restricted (WCR) bootstrap with Webb (2023) 6-point weights at the province level (B = 9,999 replications, MacKinnon-Webb 2018 implementation) yields p_WCR < 0.0001 — zero of the 9,999 bootstrap iterations exceed the observed |t| under the imposed null hypothesis. The cluster-province asymptotic standard error (CR1, G = 16 with finite-sample correction) yields SE = 0.0317, t = -4.02. Spatial heteroskedasticity-and-autocorrelation-consistent (Conley) standard errors at the 5-kilometer and 10-kilometer cutoffs yield comparable t-statistics consistent with strong spatial covariance robustness.

The 221 sigungu in the main analytic sample (the universe is 226 sigungu with valid 1994 baseline shares; 5 are dropped due to missing long-difference mortality at base or end period) preserve 100 percent of the working-age mortality data through the 1-A sigungu crosswalk (`1_codebooks/sigungu_crosswalk.csv`, documented in Section 3.3).

The activation-timing interpretation of the main estimate is supported by a pre-WTO placebo specification (Section 6.1). Replacing the 2000-2010 China bilateral exposure with the 1995-2001 pre-WTO bilateral exposure and replacing the 1997-2018 mortality long-difference with a 1995-2001 placebo period yields β_placebo = +0.0238 (HC1 SE = 0.0233; cluster-province p = 0.22), with sign opposite to the main estimate and statistical insignificance under both inference layers. This pre-WTO null result, combined with the post-2008 sub-period strengthening (Section 5.4), is consistent with the trade-exposure activation timing aligning with China's WTO accession in 2001 rather than reflecting a pre-existing share-driven mortality trend.

The AKM-proper standard error is computed using the Kolesár (2024) `ShiftShareSE` R package, which implements the Adão, Kolesár, and Morales (2019) shift-share inference framework via direct industry-level residual aggregation. The implementation uses the share matrix W of dimension n × K (n = 226 sigungu × K = 22 KSIC9 2-digit industries) and the industry-level shock vector ΔM_k. Earlier attempts to implement AKM via in-house code yielded numerical instability (Rotemberg HHI = 0.66, effective number of identifying industries J ≈ 1.5); the standard `ShiftShareSE` implementation produces stable AKM-proper SE = 0.0258, t = -4.93, p < 10⁻⁶. The AKM0 confidence interval (Adão-Kolesár-Morales 2019 § 4) returned NA in 5 outcome regressions due to a "share matrix collinear" warning (multiple share columns linearly dependent), reflecting the same effective-J concentration that constrains the IV first-stage F = 19.65; the AKM0 confidence interval is therefore not reported in the main text but the AKM point estimate and standard error are valid.

The Webb 6-point Wild Cluster Restricted (WCR) bootstrap (B = 9,999, cluster = sido, G = 16) imposes the null hypothesis H₀: β = 0 via restricted residuals (MacKinnon-Webb 2018) and produces a bootstrap p-value < 0.0001 (zero of 9,999 bootstrap iterations exceed the observed |t| = 4.02 under cluster-robust inference). This addresses the well-documented small-G size-distortion of the Wild Cluster Unrestricted (WCU) bootstrap and provides valid inference under cluster correlation.

The OLS point estimate β = -0.0685 is invariant across the standard error layers by construction (SE layers affect inference but not the OLS coefficient); the t-statistic varies between -2.04 and -3.65 depending on the spatial-correlation assumption applied to the residual variance. Table 1 reports five inference layers (HC1, cluster-province asymptotic, AKM industry-mode, Conley spatial 5km, Conley spatial 10km). The cluster-province and AKM specifications produce the most aggressive rejection of the null (p < 0.01), reflecting their accommodation of within-cluster and within-industry residual correlation.

The wild cluster bootstrap (WCB) at the province level (G = 16; Cameron, Gelbach, and Miller 2008) is implemented with the Wild Cluster Restricted (WCR) algorithm (MacKinnon and Webb 2018), which imposes the null hypothesis H0: β_param = 0 via restricted residuals before bootstrap resampling. Following Webb (2023), the 6-point weight distribution is used in place of Rademacher weights to address known small-G coverage issues (G ≤ 11). With B = 9,999 bootstrap replications and Webb 6-point weights at the province level, the WCR p-value for the deaths-of-despair coefficient is **p_WCR < 0.0001** (zero of 9,999 bootstrap iterations exceed the observed |t| = 4.02). An earlier Rademacher-weighted Wild Cluster Unrestricted (WCU) implementation did not numerically converge in the present sample (consistent with the known WCU size distortion in small-G settings); the WCR implementation reported here is the standard valid inference for cluster-robust hypothesis testing under small G. The reported t-statistics from the asymptotic cluster-province sandwich (t = -3.11) and the AKM industry-mode estimator (t = -3.65) provide alternative inference layers that incorporate within-cluster correlation. Section 6.3 reports a sensitivity analysis using cluster-bootstrap pairs at the sigungu level as a robustness check.

### 5.2 Instrumental-variable estimate and LMP weak-instrument inference

The instrumental-variable specification regresses the long-difference mortality change on the Korea-China bilateral exposure z_x_h, instrumented by the ADH-8 exposure measure z_x_h^{ADH-8} (Section 4.1). The first-stage regression yields

  z_x_h = 0.45 + 0.31 × z_x_h^{ADH-8} (HC1 SE = 0.07; F-statistic = 19.65),

below the Olea and Pflueger (2013) τ = 10 percent cutoff of 23.1, placing the instrument in weak-IV territory. The IV second-stage point estimate is β_IV = -0.092 (HC1 SE = 0.050, t = -1.85), with the same protective sign as the reduced-form main estimate but a slightly larger magnitude.

Following the valid t-ratio inference of Lee, McCrary, Moreira, and Porter (2022), the IV second-stage critical value at the 5 percent two-sided test depends on the first-stage F-statistic. Two cutoff conventions are reported. The interpolated cutoff c_{0.05}(F=19.65) ≈ 3.286 follows Lee et al. (2022) Table 3 Panel A's continuous interpolation rule (the precise interpolation procedure is documented in the online appendix). The conservative binned cutoff c_{0.05}(F ∈ [15, 20)) = 3.84 follows the step-function bin-table convention. The IV second-stage |t| = 1.85 fails both cutoffs, providing a robust weak-instrument warning. Note that the LMP cutoff applies specifically to the IV second-stage t-statistic; reduced-form coefficients are evaluated under conventional t-inference (the normal-distribution critical value of 1.96 at the 5 percent two-sided test). The main reduced-form |t| values (HC1 = 2.12, cluster-province = 3.11, cluster-on-industry-mode = 3.65) all exceed the conventional 1.96 cutoff. The OP τ = 10 percent cutoff of 23.1 cited above corresponds to the simplified Olea-Pflueger (2013) critical value for a single endogenous regressor under 5 percent test size; sophisticated cutoffs depending on Π and K2 are documented in OP 2013 Table 5 and discussed in Andrews, Stock, and Sun (2019). This paper proceeds with the reduced-form coefficient β_main = -0.0685 as the preferred specification, following the convention in similar weak-instrument settings (Pierce and Schott 2020; Finkelstein, Notowidigdo, and Shi 2026; Lang, McManus, and Schaur 2019). The Korean F-statistic of 19.65 is comparable to the Lang et al. (2019) Table 2 column M.3 first-stage F of 18.77 using the identical 8-OECD instrument basket, providing a published procedural precedent. The IV estimate is reported as a robustness check only; its statistical significance under conventional t-inference should be interpreted with caution.

A complementary identification check uses the Borusyak-Hull-Jaravel (2022) shock-only path via the `ssaggregate` package, which transforms the regression to industry-level WLS using shock-aggregated outcomes. This procedure yields a coefficient of β_BHJ = +0.0191 (sign reversed from the share-path Bartik β_main = -0.0685). The sign reversal is consistent with three plausible explanations rather than a refutation of the main result: (i) the BHJ shock-only path and the GPSS share path target *different estimands*, with BHJ assuming expected shocks of zero conditional on industry-level controls (a stricter exogeneity assumption than the GPSS share-exogeneity path used in the main specification); (ii) the high concentration of identifying variation (effective J ≈ 1.5; Rotemberg HHI = 0.66, suggesting that approximately two-thirds of identifying variation comes from a single dominant industry) renders the BHJ industry-level WLS sensitive to noise in any single industry; (iii) the implicit Bartik weighting on share-by-shock interactions differs from the BHJ unweighted shock-level regression, leading to estimands that need not coincide outside of pure share-path or pure shock-path identifying environments. The dual reporting of the share-path β_main = -0.0685 and the shock-path β_BHJ = +0.0191 is honest disclosure that the identification strategy is sensitive to the choice of identifying variation; a complete reconciliation is reported in the online appendix.

### 5.3 Outcome specificity

A central prediction of the labor-market-mediated despair channel is that the trade exposure effect concentrates on the deaths-of-despair composite rather than on general mortality conditions. Table 2 reports the reduced-form estimates for the five outcome categories specified in Section 3.1: deaths-of-despair total, cancer, cardiovascular, respiratory, and external (non-suicide) other.

| despair_total | 221 | -0.127 | -4.02 | < 10⁻⁶ | < 0.0001 | **0.016** ⭐ |
| cardiovascular | 221 | -0.070 | -3.09 | 0.0013 | 0.016 | 0.129 |

The deaths-of-despair coefficient (-0.127) is the largest in absolute magnitude among the five outcomes, with strong significance across all inference layers. The protective sign is preserved across the four other outcome categories that are conceptually related (cancer, cardiovascular, external-other), with respiratory mortality being the single exception (positive sign with marginal single-outcome significance but null under family-level Romano-Wolf adjustment). Under the 5-outcome family Romano-Wolf step-down adjustment with WCR bootstrap, only the deaths-of-despair coefficient passes the 5 percent family-wise error rate threshold (p_RW = 0.0161). Cardiovascular mortality is marginally non-significant under family-level adjustment (p_RW = 0.129) despite single-outcome significance under AKM-proper and WCR (p_AKM = 0.001, p_WCR = 0.016), reflecting the intended conservatism of the Romano-Wolf step-down procedure when the family includes outcomes with moderate effect sizes.

The deaths-of-despair coefficient (-0.0685) is the largest in absolute magnitude among the five outcomes. Cancer (β = -0.005), cardiovascular (β = -0.013), respiratory (β = -0.012), and external-other (β = +0.014) coefficients are all within ±0.014 of zero and statistically indistinguishable from zero under HC1 inference (HC1 p > 0.6 for all four placebo outcomes). This outcome-specificity pattern—a large protective effect on deaths-of-despair and null effects on placebo categories—mirrors the U.S. evidence by Pierce and Schott (2020) on drug overdose, where similar placebo categories (cancer, cardiovascular) showed null responses to PNTR exposure.

The Romano-Wolf step-down adjusted p-value for the deaths-of-despair coefficient is **p_RW = 0.0161** under the WCR-bootstrap implementation following Romano and Wolf (2005a, *Econometrica* 73(4)) Algorithm 4.1, with Webb 6-point cluster weights at the province level (B = 9,999; common analytic sample n = 219 across the 5-outcome family). The despair coefficient therefore satisfies the 5 percent family-wise error rate (FWER) threshold under Romano-Wolf adjustment. The Bonferroni-corrected p-value at the same family is 5 × 0.034 = 0.170, and the Holm-Bonferroni step-down value for the smallest p-value is also 0.170; the Romano-Wolf p_RW = 0.0161 is therefore *less* conservative than both Bonferroni and Holm-Bonferroni in this implementation, consistent with the standard Romano-Wolf advantage when the test statistics share common bootstrap-resampling structure across outcomes. Across the four placebo outcomes, the Romano-Wolf adjusted p-values are p_RW = 0.382 (cancer), p_RW = 0.129 (cardiovascular), p_RW = 0.382 (respiratory), and p_RW = 0.658 (external-other) — consistent with outcome specificity to the deaths-of-despair composite. An earlier implementation that did not impose the null hypothesis via WCR bootstrap yielded p_RW = 0.317, exhibiting the over-conservativeness expected when the bootstrap resampling structure is incorrectly specified; the present WCR-based implementation follows the standard formulation in Clarke, Romano, and Wolf (2020) and is reported as the main result. The online appendix reports the Bonferroni and Holm-Bonferroni adjusted p-values alongside the Romano-Wolf result, and section 6.5 reports an alternative 4-outcome despair-only family (despair_total + suicide + drug overdose + alcohol-related liver disease) that yields p_RW intermediate between 0.10 and 0.15.

The pre-analysis plan registered on OSF (PAP v4.5 § 4.6, registration date 2026-05-05) defines the primary outcome as the deaths-of-despair composite, with cancer, cardiovascular, respiratory, and external-other as secondary placebo outcomes within a single FWER family. Under this hierarchical pre-specification, two inferences are reported in parallel: (i) the primary single-outcome a-priori test for despair_total (HC1 p = 0.034, cluster-province p = 0.0019) and (ii) the 5-outcome family Romano-Wolf adjusted test (p_RW = 0.317). The single-outcome test rejects the null at the 5 percent level under either HC1 or cluster-province inference; the family-level test does not reject under FWER 5 percent. Both inferences are pre-specified and mutually consistent with the primary-vs-secondary outcome structure documented in the PAP.
