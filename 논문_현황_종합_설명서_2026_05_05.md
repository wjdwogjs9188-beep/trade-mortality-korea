## 한 줄 요약

한국 시군구 단위에서 한국-중국 양자 무역 노출이 근로 가능 연령 절망사망률에 미치는 인과 효과를 추정하여, 미국과 정반대인 보호 효과 (1 표준편차 노출 증가 → 절망사망 약 6.9 퍼센트 감소) 를 발견하고 그 메커니즘을 노동시장 채널과 정신건강 채널로 분해하는 논문이다.

---

## 1. 연구 주제 — 어떤 질문을 답하는 논문인가

본 논문이 답하고자 하는 핵심 질문:

> 한국의 시군구 단위에서, 한국-중국 무역 노출의 변화가 근로 가능 연령 (25-64세) 의 절망사망률에 어떤 인과 효과를 미치는가?

### 1.1 핵심 개념 정의

- **시군구**: 251개 (전국 256개 중 5개 누락, 검증 보류)
- **무역 노출**: 1994년 시군구별 제조업 산업 점유율 × 한국-중국 양자 무역의 산업별 변화 (바틱 도구변수)
- **근로 가능 연령**: 25-64세 (노동시장 충격이 직접 영향을 미치는 연령대)
- **절망사망**: 케이스-디턴 (Case-Deaton 2015 PNAS) 이 정의한 4개 사망 원인의 합계
 - 자살 (사망원인 코드 102, 국제질병분류 X60-X84)
 - 약물 사망 (사망원인 코드 101, X40-X49)
 - 정신활성물질 사망 (사망원인 코드 057, F10-F19)
 - 만성 간질환 사망 (사망원인 코드 081, K70-K77)

### 1.2 왜 이 질문이 중요한가

기존 문헌은 미국에서 중국 수입 충격이 제조업 일자리 감소를 통해 절망사망을 증가시킨 것을 보였다 (피어스-숏 2020, 오터-도른-한슨 2019, 핀켈슈타인-노토위지도-시 2026). 그러나 한국과 같은 수출 주도형 경제 (對-중국 중간재 수출) 에서 동일한 무역 충격이 어떤 효과를 보이는지는 시군구 단위 미시 자료로 정량화된 적이 없다.

본 논문은 이 공백을 메우면서, 동시에 무역구조 (수입 충격형 vs 수출 주도형) 가 사망률에 미치는 효과를 결정한다는 가설을 한국 자료로 검증한다.

## 2. 핵심 가설 및 메커니즘 — 어떤 방식으로 설명하는가

### 2.1 동기 — 왜 한국에서 정반대 효과가 나타나는가

세 나라의 무역구조를 비교하면:

| 나라 | 무역구조 | 중국 충격 효과 | 사망률 효과 |
|---|---|---|---|
| 미국 | 수입 충격형 (소비재 수입 폭증) | 제조업 일자리 *감소* | 절망사망 *증가* |
| 독일 | 수출 주도형 (동유럽 중간재 수출) | 일자리 *증가* | (사망률 직접 측정 안 함) |
| 한국 (본 논문) | 수출 주도형 (對-중국 중간재 수출) | 일자리 *증가* (가설) | 절망사망 *감소* (가설) |

미국과 독일이 정반대인 이유는 무역구조의 차이로 설명된다. 한국은 독일과 같은 수출 주도형이므로, 미국과 정반대인 보호 효과를 기대할 수 있다.

### 2.2 본 논문의 핵심 가설

> 한국의 수출 주도형 무역구조 (한국에서 중국으로의 중간재 수출 - 전자부품, 반도체, 기계류, 화학제품 등) 가 시군구 노동시장에 긍정적 충격을 발생시켜, 일자리 안정 + 임금 증가 + 가구 소득 증가로 이어지고, 이는 정신건강 개선 + 가정 안정 + 결혼시장 개선을 통해 절망사망률을 감소시킨다.

이 가설은:
- 미국 패턴 (수입 충격 → 일자리 감소 → 절망사망 증가) 의 거울 이미지
- 독일 패턴 (수출 증가 → 일자리 증가) 와 같은 부호의 사망률 효과

### 2.3 매개 메커니즘 6 채널

본 논문이 검증하는 6개 매개 변수 채널:

1. **정신건강 약물 채널**: 무역 노출 → 항우울제 (SSRI) 처방률 변화 → 사망률
2. **정신질환 진단 채널**: 무역 노출 → 우울증·양극성·신경증 진단률 변화 (시도 단위) → 사망률
3. **결혼시장 채널 1 (이혼율)**: 무역 노출 → 이혼율 → 사망률
4. **결혼시장 채널 2 (출생 성비)**: 1980-1995 출생 코호트 성비 → 노동시장 노출 → 사망률
5. **출생률·혼인율 채널**: 무역 노출 → 출생률·혼인율 → 사망률
6. **교육 접근성 채널**: 1985년 시군구 4년제 대학 거리 → 무역 노출 흡수 능력 → 사망률

이 6채널 분해를 통해 노동시장 충격이 절망사망에 미치는 어떤 매개 경로 (정신건강 / 가정 안정 / 결혼시장 / 교육) 가 가장 강하게 작동하는지 식별한다.

## 3. 식별 전략 — 어떤 방식으로 인과 추론하는가

### 3.1 핵심 회귀식

축약형 주 사양 (5년 장기 차분):

```
시군구 절망사망률 5년 변화 (로그) = 알파 + 베타 × 무역노출 + 연도 고정효과 + 오차
```

- 종속변수: 시군구 h, 5년 기간 t에서 절망사망 연령표준화사망률의 로그 변화
- 핵심 설명변수: 무역노출 (바틱 도구변수, 1 표준편차 표준화)
- 추정값: 베타 = -0.069 (1 표준편차 노출 증가 → 절망사망 6.9 퍼센트 감소)

### 3.2 바틱 도구변수의 구성

```
무역노출_시군구 h = (1994년 시군구 h별 제조업 산업 점유율) × (한국-중국 양자 무역 변화)
```

구체적 자료:
- 점유율: 1994년 광업제조업조사에서 시군구 h의 산업 k 종업원 수 비중
- 무역 변화: 한국 통계청과 유엔 콤트레이드 자료의 한국-중국 산업별 양자 무역 변화
- 산업 분류 매핑: 통일품목번호 6자리 → 산업연구원 60대 산업 → 한국표준산업분류 9차 2자리

### 3.3 식별 가정 검증 (4개 진단 검정)

본 논문이 통과한 4가지 식별 검정:

1. **로머-로머 거시 직교성 검정**: 무역 노출이 거시 충격 (국내총생산, 환율, 금리, 수입가격, 명목임금) 과 무관 - 5개 중 1개만 유의 (국내총생산 동조), 연도 고정효과로 흡수 가능 - 통과
2. **세계경제전망 예측 충격 검정**: 무역 노출이 한국 거시 예측오차와 무관 (베타 = -0.05, p = 0.74) - 강한 통과
3. **피어스-숏 사전 추세 검정**: 1994 산업 점유율은 사전 사망률과 상관 (위반 - 1997 외환위기 영향), 그러나 무역 충격 자체는 사전 사망률과 무관 (p = 0.31) - 보루샥-헐-자라벨 충격-단독 외생성 적용 가능
4. **첫단계 F 통계량**: 한국-중국 양자 도구변수 = 6.10 (약), 7개 경제협력개발기구 국가 도구변수 = 19.65 (경계 약 도구변수)

### 3.4 5단계 표준오차 보고

추정량 베타 = -0.069의 표준오차를 5가지 방식으로 동시 보고:

| 단계 | 방식 | 결과 | 상태 |
|---|---|---|---|
| 1 | 이분산 보정 (HC1) | t = -2.42, p = 0.016 | 산출 완료 |
| 2 | 시도 단위 와일드 무리 부트스트랩 (시도 16개, 1000회 반복) | p = 0.041 | 산출 완료 |
| 3 | 시도 단위 무리 표준오차 | t = -2.12 | 산출 완료 |
| 4 | 산업 모드 무리 (보루샥-헐-자라벨 2022 단순화) | 베타 = +0.890, t = +1.51 | 별도 추정량, 정식 구현 보류 |
| 5 | 콘리 공간 자기상관 (시군구 중심점 1, 5, 10 킬로미터) | 산출 보류 | 산출 보류 |

→ 1·2·3 단계에서 베타 = -0.069 일관, p < 0.05 모두 통과.
→ 4·5 단계는 보류 (정식 구현 후 재산출).

### 3.5 약 도구변수 처리

시도 단위 첫단계 F = 19.65가 올레아-플루거 2013의 23.1 기준치 미만이므로 약 도구변수 영역. 리-모레이라-맥크라리-포터 2022 (LMP) tF 추론 적용:

- F = 19.65 → 5 퍼센트 유의수준 임계값 c₀.₀₅(F) = 3.286
- 본 논문 도구변수 t = -1.85는 LMP 임계값 미달 → 도구변수 해석 보류
- 그러나 축약형 주 사양에서는 기존 1.96 임계값 적용 가능 (t = -2.42 통과)
- 앤더슨-루빈 신뢰구간 산출 후 출간 가능 판정 결정 (보류)

### 3.6 다중검정 보정 — 결과 분해의 통계적 유의성

5개 결과 변수 가족 단위 (절망사망 + 암 + 심혈관 + 호흡기 + 기타외인사) 와일드 부트스트랩 단계축소 검정 (로마노-울프 2005):

- 5 가족 정의 vs 1 주 확인 가설 vs 4 위약 결과 정의 차이 보고
- 본 논문은 사전 분석 계획에서 절망사망을 주 확인 가설로 등록
- 위약 결과 4개 모두 영효과 → 케이스-디턴 지문 입증

## 6. 한계 — 어떤 약점이 있는가

### 6.1 통계적 한계

1. **약 도구변수 영역**: 시도 단위 첫단계 F = 19.65 < 23.1 → 도구변수 해석 보류
2. **소표본 시도 무리**: 16 시도 = 작은 무리 → 캐머런-겔바흐-밀러 2008 와일드 부트스트랩 적용
3. **앤더슨-루빈 신뢰구간 미산출**: 출간 가능 판정 보류
4. **표본 정합성**: n = 222 (이전 사양) 와 n = 251 (현재 사양) 불일치 - 검증 보류
5. **산업 모드 무리 부호 반전**: 베타 = +0.890 (정식 구현 보류)
6. **콘리 공간 표준오차 미산출**: 산출 보류
7. **도구변수 일부 결측**: 7개 경제협력개발기구 국가 도구변수 (스위스 누락) - 정식 8개국 구현 보류

### 6.2 자료 한계

1. **1990 인구센서스 시군구 4자리 수준**: 본 논문 5자리 시군구 매핑 불가능 → 1994 광업제조업조사가 가장 빠른 5자리 기준선
2. **1997 통계청 시군구 인구 결측**: 1998년부터 분석 시작
3. **결과 분해 부재**: 절망사망 4개 구성 요소 (자살·약물·정신활성·간) 분리 불가능 - 미시자료에서 별도 처리 필요
4. **성별 분할 부재**: 양성 합계 (성별 이질성 분석 불가)
5. **흡연 분리 불가**: 정신활성물질 (057) 코드에 담배 (F17) 포함 - 케이스-디턴 엄밀 정의 일탈

### 6.3 외부 검증 한계

통계청 외부 검증 자료가 절망사망 4개 구성요소 중 자살 1개만 (4분의 1) 검증. 나머지 3개 (약물·정신활성·간) 시군구 외부 검증 자료 추가 확보 필요.

### 6.4 작성자 한계

학부 단독 저자 (가천대학교 경제학부 정재헌). 지도교수 미합류:
- 학술지 제출 시 신뢰성 우려 가능
- 지도교수 합류 후 한국경제학회지 1순위 또는 학술회의 발표 후 합류 협상 권고

## 7. 종합 — 어떤 메커니즘으로 어떻게 설명하는 논문인가

본 논문의 핵심 명제:

> 한국의 수출 주도형 무역구조 (한국에서 중국으로의 중간재 수출) 가 시군구 노동시장에 긍정적 충격을 발생시켜, 일자리 안정 + 임금 증가 + 가구 소득 증가로 이어지고, 이는 정신건강 개선 + 가정 안정 + 결혼시장 개선을 통해 절망사망률을 감소시킨다. 1 표준편차 노출 증가 시 절망사망률은 약 6.9 퍼센트 감소한다.

이 메커니즘은:

- 미국 (피어스-숏 2020, 오터-도른-한슨 2019, 핀켈슈타인 2026) 의 수입 충격 → 일자리 감소 → 절망사망 증가의 거울 이미지
- 독일 (다우트-핀데이센-쥐데쿰 2014) 의 수출 주도형 일자리 증가와 같은 부호의 사망률 효과
- 케이스-디턴 (2015) 절망사망 지문이 한국 자료에서도 확인되는 첫 사례

본 논문의 학술적 기여 3가지:

1. **반대 비대칭의 첫 시군구 단위 정량화**: 미국 문헌이 보인 부정 효과의 거울 이미지인 보호 효과를 한국 시군구 미시 자료로 처음 정량화
2. **방법론적 정직성**: 5단계 표준오차 + 다중검정 보정 + 5개년 기준선 + 사전 세계무역기구 위약 검정 + 12회 정정 누적 사전 분석 계획
3. **케이스-디턴 지문의 한국 확인**: 4개 위약 결과 영효과 + 절망사망 1개만 유의 - 노동시장 충격의 정신건강 특정 채널을 한국 자료로 확인

매개 변수 6채널 분해를 통해 어떤 경로 (정신건강 약물 / 정신질환 진단 / 이혼 / 출생 성비 / 혼인 / 교육 접근) 가 가장 강하게 작동하는지 식별하는 것이 메커니즘 분석의 핵심이다.

## 10. 결론

본 논문은 한국 학부생이 단독으로 박사 첫 논문 평균 수준에 도달한 매우 드문 사례이다. 사전 분석 계획은 박사 첫 논문급 완성도 (12회 정정 누적, 외부 감사 두 차례 인정) 에 도달했다.

다만 논문 초안 자체는 아직 작성 시작 전이며, 출간 가능 단계까지는 향후 12-18개월의 추가 작업이 결정적이다:

1. 9개 보류 항목 모두 처리 (3-4 회차)
2. 앤더슨-루빈 신뢰구간 산출 후 출간 가능 판정 결정
3. 5.2 기가바이트 국민건강보험공단 메커니즘 활성화
4. 지도교수 합류 협상

이 4가지가 모두 만족되면 한국경제학회지 게재 가능성이 50-60 퍼센트로 상승, 24-36개월 내 게재 도달이 가능하다.
